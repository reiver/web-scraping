Web Scraping

making the human-legible web machine-legible

by Charles Iliya Krempeaux

Web Scraping is the programmatic extraction of specific human-legible (but not machine-legible) data from web pages that is for alternative use or storage.

The intuition might be that — web scraping involves having a machine read a user interface (UI) solely designed and intended for humans.

So, for example, reading HTML that does NOT have: JSON-LD, Microdata, Microformats, RDFa, or any other machine-legible formats.

Screen Scraping

The term web scraping evolved from the term screen scraping.

Screen Scraping is the process of programmatically reading text from a computer terminal's screen.
